# CogniMail - ML-Powered Anti-Phishing and Spam Filtering

Enterprise email security system with ML-based phishing/spam detection, Gmail integration, and real-time monitoring.

## Tech Stack
- **Frontend:** React, CSS Modules
- **Backend:** Django, Celery
- **ML:** Scikit-learn, TensorFlow, SpamAssassin
- **Infra:** Docker, PostgreSQL, Redis
